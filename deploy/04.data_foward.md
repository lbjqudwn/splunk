# 1. 데이터 포워딩

forwarder를 통해서 데이터를 인덱서 클러스터에 포워딩해보자.  
테스트는 universal forwarder를 사용하며, 포워더의 splunkd.log를 포워딩한다.  
아울러 포워더 관리를 위한 deployment server 설정도 해보자.  

- 포트 정보  

인스턴스|용도|포트|비고
---- | ---- | ---- | ----
uf_1|mgmt port|8089|
uf_2|mgmt port|8089|
ds|web port|8200|deployment server
----|mgmt port|8289|
----|appserver port|8265|
----|kvstore port|8391|
