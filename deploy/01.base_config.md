# 2. 스플렁크 아키텍쳐

분산배포 환경으로 서치헤더 클러스터, 인덱서 클러스터를 구성한다.
관리서버는 스플렁크 아키텍처의 전반적인 관리를 담당하며, Cluster Master, Deployer, Deployment Server, License Master, Monitoring Console로 구성된다.
